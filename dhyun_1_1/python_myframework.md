# 웹 프레임워크란 무엇인가?

Python 위키에 따르면, 웹 프레임워크란 웹앱이나 웹서비스를 프로토콜, 소켓 신경쓰지 않고 구현하기 위한 패키지의 모음이다.
[<sup>1</sup>](https://wiki.python.org/moin/WebFrameworks)
